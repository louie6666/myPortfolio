# Personal Portfolio | Louie Lusterio

A clean, minimalist personal portfolio website featuring a dynamic dark/light mode toggle, responsive design, and a modern UI.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
- **Icons:** [Lucide Icons](https://lucide.dev/), [Font Awesome](https://fontawesome.com/)
- **Fonts:** Inter (via Google Fonts)
- **Deployment:** [Netlify](https://www.netlify.com/)

## Key Features
- **Dynamic Theme Switching:** Seamless transition between dark and light modes using CSS variables.
- **Responsive Grid:** A custom-built layout that adapts to different screen sizes.
- **Vibe Coding Energy:** Showcasing a journey from a diploma-seeker to a passionate developer.
- **Interactive Elements:** Hover effects, smooth scrolling, and an education timeline.
